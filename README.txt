# SCADA-эмулятор задвижки

## Описание

Проект представляет собой программный эмулятор задвижки НПС
с визуальным SCADA-интерфейсом.

Система моделирует движение задвижки, работу датчиков и режимы отказов.

---

## Возможности

- Управление положением задвижки (0–100%)
- Режимы работы:
  - normal
  - jammed
  - drive_failure
- Поддержка вертикальной и горизонтальной ориентации
- Падение вертикальной задвижки при отказе привода
- Датчик положения с фильтрацией
- Реалистичная модель температуры
- История команд (Command Pattern)
- Журнал аварий
- SCADA-подобный интерфейс
- Графики параметров в реальном времени

---

## Структура проекта

models.py — логика задвижки
commands.py — команды управления
gui.py — интерфейс SCADA
main.py — запуск программы

---

## Основные параметры

- **Target Position** — заданное положение
- **Actual Position** — фактическое положение
- **Measured Position** — положение по датчику

---

## Режимы

**normal** — штатная работа
**jammed** — заклинивание
**drive_failure** — отказ привода

Для вертикальной задвижки при отказе привода происходит падение вниз.

---

## Температура

Температура рассчитывается на основе:

- температуры окружающей среды
- движения задвижки
- нагрузки механизма
- аварийных режимов

После остановки происходит постепенное охлаждение.

---

## Запуск

Установить зависимости:

```bash
pip install matplotlib
```

Запустить:

```bash
python main.py
```
